# AI Productivity Applications (Under Development)

## Overview

This repository contains a collection of web-based applications focused on improving productivity, learning, and user interaction through modern UI design and browser-based intelligence features.

The project is currently under active development and includes multiple independent applications built using HTML, CSS, JavaScript, and React.

---

## Applications Included

### 1. AI Text Summarizer

A web interface designed to generate concise summaries from large text inputs.

**Features:**

* Multiple summary formats (bullet points, paragraph, numbered)
* Adjustable summary length
* Word count tracking
* Copy and download functionality
* Voice input support
* Text-to-speech output

---

### 2. AI Student Counselor

An interactive chat-based interface that simulates academic and general student guidance.

**Features:**

* Conversational UI
* Real-time responses
* Voice interaction capability
* Clean and responsive design

---

### 3. Voice Notepad To-Do Application

A task management system enhanced with voice commands and speech feedback.

**Features:**

* Add, complete, and delete tasks
* Voice command support
* Text-to-speech responses
* Real-time task updates
* Simple notepad-style interface

---

## How to Run

### HTML Application

1. Download or clone the repository
2. Open the HTML file directly in a browser

### React Components

1. Create a React project (using Vite or Create React App)
2. Add the `.tsx` files into the `src` directory
3. Install dependencies:

   ```
   npm install
   npm install lucide-react
   ```
4. Run the development server:

   ```
   npm run dev
   ```
